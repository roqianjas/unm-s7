## ğŸ› ï¸ TOOLS UNTUK MEMBUAT KUESIONER

### 1. Google Forms (Recommended)
**Kelebihan:**
- Gratis dan mudah digunakan
- Otomatis tersimpan di Google Drive
- Analisis otomatis (chart, summary)
- Bisa diakses via QR Code
- Export ke Excel/CSV

**Cara Membuat:**
1. Buka forms.google.com
2. Klik "+ Blank" untuk form baru
3. Copy-paste pertanyaan dari dokumen ini
4. Atur tipe pertanyaan (multiple choice, checkbox, paragraph, dll)
5. Aktifkan "Required" untuk pertanyaan wajib
6. Customize theme (warna, font)
7. Preview dan test
8. Dapatkan link dan QR Code

**Tips:**
- Gunakan "Section" untuk membagi bagian A-E
- Aktifkan "Limit to 1 response" jika perlu
- Gunakan "Response validation" untuk email
- Buat "Thank you message" yang menarik

### 2. Microsoft Forms
**Kelebihan:**
- Terintegrasi dengan Office 365
- Interface mirip Google Forms
- Export ke Excel

**Cara Akses:**
- forms.office.com (perlu akun Microsoft)

### 3. Typeform
**Kelebihan:**
- UI/UX sangat menarik
- One question at a time (conversational)
- Animasi dan interaktif

**Kekurangan:**
- Free plan terbatas (10 pertanyaan, 100 respons/bulan)
- Perlu upgrade untuk fitur lengkap

### 4. SurveyMonkey
**Kelebihan:**
- Fitur analisis advanced
- Template profesional

**Kekurangan:**
- Free plan sangat terbatas
- Perlu bayar untuk fitur lengkap

**Rekomendasi**: Gunakan **Google Forms** karena gratis, mudah, dan cukup untuk kebutuhan akademik.

## ğŸ“± CARA MEMBUAT QR CODE

### Online QR Code Generator (Gratis)

1. **QR Code Generator** (qr-code-generator.com)
   - Paste link Google Forms
   - Pilih "URL"
   - Customize warna (opsional)
   - Download PNG/SVG

2. **QR Code Monkey** (qrcode-monkey.com)
   - Fitur lebih lengkap
   - Bisa tambah logo di tengah
   - High resolution

3. **Google Chrome Extension**
   - Install "QR Code Generator"
   - Klik icon saat di halaman Google Forms
   - Instant QR Code

### Tips QR Code:
- **Size**: Minimal 3x3 cm di X-Banner (agar mudah di-scan)
- **Contrast**: Hitam di background putih (paling reliable)
- **Test**: Scan dengan berbagai device sebelum cetak
- **Backup**: Sediakan link pendek juga (bit.ly/curheart-survey)
