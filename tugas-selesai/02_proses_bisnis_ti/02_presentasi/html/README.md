# Presentasi Proses Bisnis TI - SATRIAMART

Presentasi HTML untuk mata kuliah Proses Bisnis TI tentang Implementasi Proses Bisnis dengan Analisis SWOT pada SATRIAMART.

## Daftar Slide

1. **slide-01-judul.html** - Halaman judul presentasi
2. **slide-02-latar-belakang.html** - Latar belakang penelitian
3. **slide-03-rumusan-masalah.html** - Rumusan masalah
4. **slide-04-tujuan-penelitian.html** - Tujuan penelitian
5. **slide-05-manfaat-penelitian.html** - Manfaat penelitian
6. **slide-06-metode-penelitian.html** - Metode penelitian
7. **slide-07-profil-perusahaan.html** - Profil SATRIAMART
8. **slide-08-proses-bisnis.html** - Proses bisnis berjalan
9. **slide-09-analisis-swot.html** - Analisis SWOT
10. **slide-10-kendala.html** - Kendala yang dihadapi
11. **slide-11-solusi.html** - Solusi yang diusulkan
12. **slide-12-kesimpulan.html** - Kesimpulan dan saran
13. **slide-13-terima-kasih.html** - Penutup

## Cara Menggunakan

1. Buka setiap file HTML di browser (Chrome, Firefox, Edge, dll)
2. Setiap slide adalah file terpisah untuk memudahkan navigasi
3. Gunakan mode presentasi browser (F11) untuk tampilan fullscreen
4. Presentasi menggunakan Tailwind CSS untuk styling modern dan responsif

## Teknologi

- HTML5
- Tailwind CSS (via CDN)
- Desain modern, clean, dan minimalist
- Responsif untuk berbagai ukuran layar

## Catatan

- Presentasi dibuat berdasarkan makalah lengkap yang ada di folder `01_laporan`
- Desain menggunakan palet warna netral (slate) untuk tampilan profesional
- Setiap slide fokus pada poin-poin penting tanpa detail berlebihan
- Bahasa Indonesia sesuai KBBI dan EYD
