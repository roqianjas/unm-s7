# MAKALAH PROSES BISNIS TI - SATRIAMART

## Implementasi Proses Bisnis dengan Analisis SWOT pada SATRIAMART

### Deskripsi
Makalah ini merupakan hasil analisis komprehensif terhadap proses bisnis SATRIAMART, sebuah perusahaan manufaktur dekorasi dan aksesoris akrilik. Analisis mencakup pemetaan proses bisnis yang berjalan, analisis SWOT, identifikasi kendala, dan rekomendasi solusi untuk optimalisasi.

### Struktur Makalah

Makalah dibagi menjadi 7 file part untuk memudahkan pembacaan:

#### **Part 1: Halaman Depan dan BAB I**
File: `makalah_proses_bisnis_satriamart_part1.md`

Konten:
- Cover/Halaman Judul
- Kata Pengantar
- Daftar Isi
- Daftar Gambar
- Daftar Tabel
- **BAB I: PENDAHULUAN**
  - 1.1 Latar Belakang
  - 1.2 Rumusan Masalah
  - 1.3 Tujuan Penelitian
  - 1.4 Manfaat Penelitian

#### **Part 2: BAB II**
File: `makalah_proses_bisnis_satriamart_part2.md`

Konten:
- **BAB II: PROFIL SINGKAT PERUSAHAAN**
  - 2.1 Gambaran Umum SATRIAMART
  - 2.2 Visi dan Misi Perusahaan
  - 2.3 Produk dan Layanan
  - 2.4 Nilai-Nilai Perusahaan
  - 2.5 Struktur Organisasi

#### **Part 3: BAB III - Bagian Proses Bisnis**
File: `makalah_proses_bisnis_satriamart_part3.md`

### Cara Menggunakan

1. **Membaca Secara Berurutan**: Baca file dari Part 1 hingga Part 7 untuk mendapatkan pemahaman lengkap
2. **Kompilasi untuk Submission**: Gabungkan semua part menjadi satu dokumen Word/PDF untuk pengumpulan
3. **Navigasi Cepat**: Gunakan struktur di atas untuk menemukan bagian spesifik yang dibutuhkan

### Cara Menggabungkan File

#### Opsi 1: Manual Copy-Paste
1. Buka file Part 1 dan copy semua konten ke Word/Google Docs
2. Lanjutkan dengan Part 2, 3, dst hingga Part 7
3. Format sesuai requirements (margin, font, spacing)
4. Export ke PDF

#### Opsi 2: Menggunakan Pandoc (jika tersedia)
```bash
pandoc makalah_proses_bisnis_satriamart_part*.md -o makalah_proses_bisnis_satriamart.pdf
```

#### Opsi 3: Menggunakan VS Code
1. Install extension "Markdown PDF"
2. Buat file baru dengan menggabungkan semua part
3. Convert ke PDF menggunakan extension

### Format Submission

Sesuai dengan syarat dalam Rencana Tugas Mahasiswa:

- **Margin**: Kiri 4 cm, Kanan/Atas/Bawah 3 cm
- **Ukuran Kertas**: A4, 70 Gram
- **Jarak Spasi**: 1,5 paragraf
- **Font Size**: 12
- **Jenis Huruf**: Arial / Times New Roman

### Struktur Makalah (Sesuai RTM)

Makalah ini mengikuti struktur 4 BAB sesuai arahan RTM:

1. **BAB I - Pendahuluan**: Latarbelakang dan permasalahan
2. **BAB II - Profil singkat Perusahaan**: Informasi tentang SATRIAMART
3. **BAB III - Pelaksanaan Kegiatan**: 
   - Proses bisnis berjalan
   - Analisis SWOT
   - Kendala yang dihadapi
   - Solusi dari kendala yang ditemukan
   - **LAMPIRAN FOTO DAN BUKTI KEGIATAN**
4. **BAB IV - Kesimpulan dan Saran**: Summary dan rekomendasi

### Catatan Penting

1. **Foto Dokumentasi**: Pada bagian 3.5 (Dokumentasi Kegiatan) dan Lampiran F, perlu ditambahkan foto-foto aktual dari kegiatan analisis seperti:
   - Foto wawancara dengan tim SATRIAMART
   - Foto observasi proses produksi
   - Foto workshop SWOT analysis
   - Screenshot sistem yang digunakan
   - Foto tim saat melakukan analisis

2. **Nama Kelompok**: Ganti placeholder "[Nama Kelompok - 3 Anggota]" dengan nama kelompok dan anggota yang sebenarnya

3. **Gambar dan Diagram**: Pada bagian yang ditandai [Catatan: ...], perlu dibuat visual diagram seperti:
   - Flowchart proses bisnis
   - Diagram SWOT
   - Matriks strategi
   - Roadmap implementasi
   - Struktur organisasi

4. **Tabel**: Semua tabel sudah dibuat dalam format markdown dan perlu diformat dengan baik di Word
