Masalah Kualitas Data

• Data duplikat (15 record)
• Missing values (8% data)
• Format tanggal tidak konsisten
• Inkonsistensi penulisan kategori
• Nilai outlier pada harga

Solusi yang Diterapkan

✓ Hapus duplikat berdasarkan ID unik
✓ Imputasi nilai kosong dengan median/mode
✓ Standarisasi format YYYY-MM-DD
✓ Normalisasi nama kategori
✓ Validasi range harga 10k-500k
