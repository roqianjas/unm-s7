## Petunjuk Penggunaan

### Navigasi
- Gunakan tombol "Selanjutnya" dan "Sebelumnya" di footer setiap slide
- Setiap slide menampilkan nomor halaman (Slide X/15)

### Konversi ke PDF
1. Buka setiap slide di browser (Chrome/Edge recommended)
2. Tekan Ctrl+P (Print)
3. Pilih "Save as PDF"
4. Set layout: Landscape
5. Set scale: 100%
6. Margins: None
7. Background graphics: On
8. Ulangi untuk semua slide

### Presentasi
- Gunakan mode fullscreen browser (F11)
- Resolusi optimal: 1920x1080 (16:9)
- Font sudah optimal untuk proyektor
