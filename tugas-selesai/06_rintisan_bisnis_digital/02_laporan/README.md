## ğŸ¯ Ringkasan Eksekutif

**Proyek**: CUR-HEART - Hypnotherapy & Mind Wellness Center  
**Jenis**: Rintisan Bisnis Digital (Healthtech)  
**Tim**: Roki Anjas (CEO), Fahruroji (CTO), Susanto (CMO)  
**Institusi**: Universitas Nusa Mandiri - Program Studi Sistem Informasi

**Problem**:
- 20% populasi Indonesia mengalami gangguan kesehatan mental
- Akses terbatas ke layanan terapi berkualitas
- Proses reservasi manual tidak efisien
- Dokumentasi terapi tidak terstruktur

**Solution**:
Platform digital terintegrasi untuk layanan hipnoterapi yang mencakup:
- Online booking system 24/7
- Integrated payment gateway
- Digital documentation system
- Multi-role dashboard (klien, terapis, admin)
- Automated notifications
- Analytics & reporting

**Market Opportunity**:
- TAM: Rp 1,3 Triliun
- SAM: Rp 134 Miliar
- SOM (Year 1): Rp 1,34 Miliar
- Market growth: 35% annually

**Business Model**:
- B2C: Pay-per-session, subscription packages
- B2B: Corporate wellness, university programs
- Training: Hypnotherapy certification

**Financial Highlights**:
- Development Cost: Rp 86,8 juta
- Year 1 Revenue: Rp 1,2 Miliar
- Break-even: Month 8
- Gross Margin: 60%
- Operating Margin: 25%

**Technology Stack**:
- Backend: Laravel 10 (PHP 8.2)
- Frontend: Blade + Tailwind CSS
- Database: MySQL 8.0
- Payment: Midtrans
- Hosting: VPS Cloud

**Key Metrics**:
- LTV/CAC: 24x
- Customer Satisfaction: 4.2/5
- System Usability: 78/100 (SUS)
- Page Load Time: 1.8 seconds
- Uptime: 99.5%
