# Presentasi CUR-HEART - Rintisan Bisnis Digital

## Deskripsi
Presentasi slide profesional untuk Capstone Project CUR-HEART dengan desain minimalis, modern, dan rasio 16:9 (standar PowerPoint).

- **Accent**: #FFB6B9 (Pink Lembut)
- **Success**: #00D26A (Hijau)
- **Warning**: #FFA94D (Oranye)
- **Info**: #667BC6 (Biru)

### Tipografi
- **Font Utama**: Poppins (Judul & Heading)
- **Font Sekunder**: Inter (Body Text)
- **Ukuran**: 48px (Judul), 18px (Body), 14px (Caption)

### Rasio Slide
- **Format**: 16:9 (1920x1080px)
- **Viewport**: Responsif dengan max-width/height

## Cara Menggunakan

### Navigasi
- **Tombol Kiri Bawah**: Kembali ke slide sebelumnya
- **Tombol Kanan Bawah**: Lanjut ke slide berikutnya
- **INDEX.html**: Halaman indeks untuk akses cepat semua slide

### Presentasi
1. Buka `INDEX.html` di browser
2. Pilih slide yang ingin ditampilkan
3. Gunakan tombol navigasi untuk berpindah slide
4. Tekan F11 untuk mode fullscreen

### Cetak/Export PDF
1. Buka slide di browser (Chrome/Edge)
2. Tekan Ctrl+P (Print)
3. Pilih "Save as PDF"
4. Atur:
   - Layout: Landscape
   - Paper size: A4 atau Letter
   - Margins: None
   - Scale: Fit to page
5. Save
